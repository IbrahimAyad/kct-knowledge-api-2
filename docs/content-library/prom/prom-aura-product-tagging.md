## How Products Are Matched

The collection pages use this matching logic:
1. Check product tags (Shopify tags)
2. Check product type
3. Check product title

If ANY aura tag matches ANY of the above, the product appears in that aura's collection.

---

## Adding New Products Checklist

When adding new prom products:

- [ ] Add appropriate aura tags from the lists above
- [ ] Products can have multiple aura tags if they fit multiple aesthetics
- [ ] Use lowercase tags for consistency
- [ ] Consider adding 2-3 tags per product for better discoverability
